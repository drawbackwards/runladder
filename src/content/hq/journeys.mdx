## Diagram conventions

Use Mermaid for flow diagrams. Example pattern:

```
<Mermaid chart={`
flowchart LR
  Start([Landing]) --> SignUp[Sign-up]
  SignUp --> Score[Score screen]
  Score --> Result{Score < 3.0?}
  Result -->|Yes| Lift[Ladder Lift CTA]
  Result -->|No| Upgrade[Pro upgrade prompt]
`} />
```
